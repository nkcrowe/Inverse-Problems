Stochastic Optimal Stopping
& Volatility Surface Calibration

A 14-week independent study implementing Longstaff-Schwartz Monte Carlo for American option pricing and Tikhonov-regularized volatility surface calibration — framed as a regularized inverse problem structurally analogous to CT image reconstruction.

Duration 14 weeks · 6 phases
Stack Python · NumPy · SciPy · Matplotlib
Context Independent · USC Applied Mathematics MS
Motivation

The Black-Scholes model assumes constant volatility — but market option prices imply a volatility that varies systematically across strikes and maturities, producing the well-known volatility smile. Recovering a consistent volatility surface from observed prices is an inverse problem: given noisy output data, infer the underlying parameters of the model.

This is structurally identical to a class of problems I encountered in biomedical engineering. In CT image reconstruction, the goal is to recover a 2D attenuation map from a set of 1D line-integral projections (the sinogram). Both problems are ill-posed in Hadamard's sense — small perturbations in the data can produce large, unstable changes in the recovered solution — and both require regularization to be tractable.

Core analogy

CT reconstruction inverts the Radon transform to recover an image from projections. Volatility calibration inverts the Black-Scholes pricing map to recover a surface from option prices. Both amplify noise through numerical differentiation, and both are stabilized by the same class of regularization techniques — Tikhonov (L2) and total variation (L1 on gradient).

Key Methods

Euler-Maruyama discretization. GBM paths are simulated using the first-order strong scheme. Validation against the analytical log-normal distribution via KS test and QQ-plot; weak convergence rate O(Δt) verified on log-log scale.

Sn+1 = Sn + μ·Sn·Δt + σ·Sn·√Δt · Zn,   Zn ~ N(0,1)
// vectorized across M paths simultaneously

Longstaff-Schwartz LSM. At each time step working backward from expiry, continuation values are approximated by OLS regression onto weighted Laguerre basis functions restricted to in-the-money paths. The exercise decision at each node follows from comparing the regression estimate to the immediate intrinsic value.

// continuation value regression at time t: C(t, S) ≈ Σᵢ aᵢ · ψᵢ(S)
β = (ΨᵀΨ)⁻¹ Ψᵀ y   // y = discounted future cashflows
// exercise if intrinsic > estimated continuation: τ = min{ t : g(St) ≥ C(t, St) }

Tikhonov calibration. The volatility surface is recovered by minimizing a penalized least-squares objective. The Jacobian is the Black-Scholes Vega matrix. The regularization term penalizes surface roughness via a finite-difference discretization of the gradient norm.

minσ ‖BS(σ) − Cmkt‖² + λ·‖∇σ‖²
// linearized solution: σ = (JᵀJ + λLᵀL)⁻¹ Jᵀ Cmkt
// J = ∂BS/∂σ (Vega), L = finite-difference gradient operator
